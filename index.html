File Encryption / Decryption

Encrypt and decrypt files using your web browser

Encrypt a File

Enter a password and drop the file to be encrypted into the dropzone below. The file will be encrypted using the password.

Minimum length eight characters, make sure it is strong!

Drag and drop the file to be encrypted into this dropzone, or click here to select a file.

Decrypt a File

Enter the password used to encrypt the file, then drop the file to be decrypted into the dropzone below.

Drag and drop the file to be decrypted into this dropzone, or click here to select a file.
